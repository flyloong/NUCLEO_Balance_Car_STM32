# Releases Notes for 702 Balance Car STM32F411 Firmware Package
 702 Balance Car for STM32F411 is a Firmware to control "702 Balance car"

##software features:
* Self balance
* Move forward and back
* Turn left and right
* Obstacle avoidance mode
* Dance mode
* Broadcast
* Rotate head
* Fall detect
